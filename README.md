## DEPENDENCIES

> NODE V22.13.1 <br>
> NPM 11.0.0

## TO RUN LOCALLY

 `npm install` <br>
 `npm run dev`

## TO DEPLOY (IF DEPLOYING TO FIREBASE)

`npm install -g firebase-tools`<br>
`firebase login` <br>
`firebase init` (hosting, create new or use existing project, dist, yes, no, no) <br>

`npm run deploy`
